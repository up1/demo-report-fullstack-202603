# Demo with Full Stack Web Development
* Fronend: React
* Backend: NodeJS with Express
* Database: SQLite


### Backend with NodeJS and Express
* REST API
* Use Express to create a REST API
* Use CORS middleware to enable CORS
* Use SQLite as the database
  * Use [sqlite in node](https://nodejs.org/api/sqlite.html)
* OpenAI API for generating responses

Create project
```
$cd backend
$npm init -y
```
Install dependencies
```
$npm install express cors openai
``` 

Start the server
```
$node index.js
```
Open http://localhost:3000 to view it in the browser

### Frontend with ReactJS and TypeScript
* React Router
Install dependencies
```
$npm install
```

Run the development server
```
$npm start
```
Open http://localhost:5173 to view it in the browser
